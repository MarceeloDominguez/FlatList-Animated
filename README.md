# Carousel of images

Animated flatlist that changes the background color according to the main image.

 ## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/FlatList-Animated.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
